How It Works

Five pipeline stages, zero partisan input.

Pipeline Overview

  1. Build precinct graph — Census TIGER 2020 VTDs as nodes; edge weights encode inverse road-network distance
  2. MCMC sampling — Weighted ReCom (DeFord, Duchin & Solomon 2021): boundary edge selection → district merge → Wilson's LERW spanning tree → population-balanced cut
  3. Hard filters — Polsby-Popper min ≥ 0.05, max population deviation ≤ 0.5%
  4. Pareto frontier — non-dominated set: maximise PP mean, minimise county splits, cut edges, max county fragmentation
  5. Extract two maps — most compact and fewest-splits plans from the frontier
The algorithm never reads partisan registration, election results, race, income, or any demographic variable.
